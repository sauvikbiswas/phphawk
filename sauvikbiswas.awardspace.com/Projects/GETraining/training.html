About the Project

The project assigned was a part of tools being developed at PPDT. The tools are used to test Experimental Engine Data. The goal was to understand and implement a Generalized Linear Model (GLM) for the observed data. Often empirical formulae are required to predict data and to predict whether variables and attributes are important to the empirical formulae or not. The test cases usually have a data set ranging from 10,000 to 1,000,000 data points; hence it was important to drive the attention towards the computation powers, the speed and the accuracy of the tool developed.

The GLM tool thus developed would be integrated into an existing analysis software developed in Java.

The Problem Statement was to develop a code, which has the ability to
a.    Compute ANCOVA tables,
b.    Generate Regression Analysis and
c.    Perform Pairwise Comparisons.


Usage of the Tool in Aerospace

The tool developed was very general in nature. It can be used in fields ranging from demography, medicine, and physical analysis to high-end engineering. Since a major portion of data variation in Aerospace is linear in nature, this tool can be used to test huge amounts of experimental data. Here are few of the uses of this tool.

  • ANCOVAGLM can be used to perform Engine Correlation Data Analysis. Engine is generally manufactured and tested at the base. The data thus collected is handed over to the customer. The customers re-validate the data by testing the engine themselves and a correlation is drawn based on the two sets of data obtained.
  • This tool can also be used in Engine Deterioration Analysis. Setting an allowable limit sets a fault margin. As time progresses the margin reduces and a prediction can be made when the engine has to be sent for servicing. This can be used to predict a trend and thus reduce the maintenance cost.
  • Power Calibration can be done by field regression. The power calibration requires many settings and the tool can accurately predict whether the setting is important or not.
  • The tool may also be used in structures to predict structural failures. CFD data can also be analyzed using this tool and factors can be predicted as a strong or weak influence to the dependent variable.
